# Advancing Cross-Organizational Interoperability Through Data Stewardship

Enabling effective data sharing across diverse systems, disciplines, and applications remains a key challenge for global science. Addressing this challenge requires more than standardization; it calls for clear, machine-actionable descriptions of stewardship practices and meaningful mappings across organizations. 

The Data Stewardship Committee (DSC) of the Earth Science Information Partners (ESIP) has begun addressing this issue by collecting and synthesizing stewardship practices across organizations and developing crosswalks among them within the context of the FAIR Guiding Principles for scientific data management and stewardship (Wilkinson et al., 2016).

This repository captures the outcomes of those activities.

# Reference
