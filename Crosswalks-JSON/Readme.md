JSON Crosswalks

This folder contains SON crosswalks from individual organizational metadata profiles to schema.org.

File Namning Conventions
Crosswalk type, source system, destination system, resource type, version, date

Version Convention: vVVrRRpPP, where v=Major Version/release, r=Revision (Minor Version), and p=Patch

Example
Metadata-Crosswalk_NASA-UMM-C_schemadotorg_dataset_v01r00p00-20260705.json
