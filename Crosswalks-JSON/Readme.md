**JSON Crosswalks**

This folder contains SON crosswalks from individual organizational metadata profiles to schema.org.

**File Namning Convention:**

crosswalkType_sourceSystem_destinationSystem_resourceType_version_creationDate.json

**Version Convention:** 

vVVrRRpPP, where v=Major Version/release, r=Revision (Minor Version), and p=Patch

**Example**

Metadata-Crosswalk_NASA-ESDIS_schemadotorg_dataset_v01r00p00-20260710.json
